Aventuras

Aplicación web responsiva que promueve el crecimiento personal mediante desafíos digitales gamificados y progresión por XP.

Vite + React Supabase Tailwind CSS shadcn-ui

El Objetivo

Desarrollar una aplicación web MVP (Producto Mínimo Viable) donde los usuarios puedan registrarse, completar retos presentados como tarjetas estilo pixel art, ganar puntos de experiencia (XP) y subir de nivel. Fomentando la participación a través de una tabla de posiciones pública competitiva.

Funcionalidades Core

Autenticación Segura

Inicio de sesión manejado vía Supabase, con perfiles de usuario que guardan nivel, progreso y fecha de registro.

Engine de Gamificación

Sistema de retos únicos. Las tarjetas voltean al hacer clic y al completarse otorgan XP automático (+100 XP = 1 Nivel).

Leaderboard Público

Tabla de posiciones accesible globalmente que lista a todos los usuarios ordenados por XP, destacando el Top 3.

Diseño Adaptable

Interfaz completamente responsiva para escritorio y móvil, con modo minimalista enfocado en la usabilidad.

Arquitectura de Datos

  • PostgreSQL (Supabase)

    Estructura relacional: `users`, `challenges` y estado intermedio `user_challenge_progress`.

  • TypeScript

    Tipado estricto para las respuestas de la base de datos y componentes UI.

El Enfoque Antropológico

"La gamificación no trata de puntos vacíos, sino de validar el esfuerzo humano. Al transformar tareas cotidianas en pixel art, reducimos la fricción cognitiva y aumentamos la dopamina basada en logros."
